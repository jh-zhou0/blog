# VMware安装CentOS7

# 1.下载

首先进入centos的镜像下载站，centos7镜像站open in new window，然后下载标准版。

# 2.安装

  1. 打开VMware，然后点击创建虚拟机，若没有，请先下载open in new window。

  1. 选择稍后安装操作系统，点击下一步。

  1. 根据需要，选择客户机操作系统、版本，点击下一步。

  1. 虚拟机名称和位置可自定义，点击下一步。

  1. 最大磁盘大小根据需要决定，默认20即可，点击下一步。

  1. 点击自定义硬件

  1. 虚拟机内存根据需要可自行调整

  1. 处理器数量和内核数量根据需要可自行调整

  1. 使用自己下载的 iso 镜像文件

  1. 网络适配器选择 NAT 模式

  1. 关闭 "硬件" 界面，点击完成

  1. 点击开启此虚拟机

  1. 点击 Install CentOS 7，等待一段时间

  1. 默认英文即可，点击 continue

  1. 选择时区为亚洲上海，点击Done

  1. 点击 SOFTWARE SELECTION，选择带图形界面的，并且选择默认配备开发工具，如gcc那些（可选）。设置好后点击 Done。

  2. 点击 NETWORK & HOST NAME，设置静态网络ip和主机名，然后点击 Done

    • 根据自己VMware的NAT的网段设置IP和网关

      • 点击，编辑 -> 虚拟网络编辑器

      • 查看VMware 子网 IP 和网关

    • 设置 IP、网关，设置 DNS Servers 为 114.114.114.114 或其他常用DNS

  3. 设置好IP和主机名后如下所示，点击Done

  4. 点击 INSRTALLATION DESTINATION，默认即可，然后点击 Done

  1. 点击 Begin Installation，开始安装

  1. 点击 ROOT PASSWORD，设置 root 账户密码，然后再创建一个普通User，然后点击Done

  1. 点击 finish configuration，完成配置

  1. 点击 reboot，重启虚拟机

  1. 重启后进入以下界面，接受许可，然后点击完成配置

  2. 点击 Not Listed？，使用root账号登录

  1. 然后一路点击 next，skip 进入centos

# 3.设置电脑 VMnet8 的IP

设置电脑 VMnet8 的IP 与 VMware 的 IP 网段一致

  1. 打开 VMnet8

  1. 点击 属性

  1. 双击 ipv4

  2. 设置 IP 和 网关
